# AppQuoteDay

Aplicativo para obter cotação em Real do Dólar, Euro e Bitcoin.

## Tecnologias usadas

* React-Native
* Typescript
* Axios

### 📋 Pré-requisitos

O que é necessário para rodar o projeto?

* [NodeJS](https://nodejs.org/en/download/package-manager)
* [Expo](https://expo.dev/)

### 🚀 Iniciar

* npm install
* npx expo start
